# Stack Menu-Driven Program in C

A menu-driven Stack implementation using an array in C.

## Features

- Push
- Pop
- Peek
- Display
- Exit
- Stack Overflow Handling
- Stack Underflow Handling

## Concepts Used

- Arrays
- Functions
- Stack (LIFO)
- Menu-Driven Programming

## Sample Menu

```
========== STACK MENU ==========
1. Push
2. Pop
3. Peek
4. Display
5. Exit
```
